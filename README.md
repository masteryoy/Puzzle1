# Puzzle1 - Block Crush

Juego tipo **match-3** inspirado en *Block Crush*.

## Cómo jugar

1. Abre `index.html` en tu navegador.
2. Haz clic en dos bloques adyacentes para intercambiarlos.
3. Forma líneas de 3 o más bloques del mismo color para sumar puntos.
4. Llega a la meta de **2500 puntos**.

## Controles

- **Reiniciar**: comienza una nueva partida.

## Tecnologías

- HTML
- CSS
- JavaScript (vanilla)
